# TIL

Today I Learned.

毎日の学習内容を短く記録するためのリポジトリです。
GitHub の草を毎日生やすことと、あとから学習を見返せる状態にすることを目的にしています。

## 運用ルール

- 1日5分以内で書く
- 長文にしない
- 箇条書きで十分
- 「詰まったこと」だけでもOK
- 毎日コミットすることを最優先にする

## ディレクトリ構成

```text
TIL/
├── README.md
├── 2026-07/
│   ├── 07-04.md
│   ├── 07-05.md
│   └── ...
├── 2026-08/
│   ├── 08-01.md
│   └── ...
```

月ごとにフォルダを作成し、その中に日付ごとの Markdown ファイルを作成します。

## テンプレート

```md
# YYYY-MM-DD

## 今日やったこと
-

## 学んだこと
-

## 詰まったこと
-

## 明日やること
-
```

## 毎日の流れ

1. VS Codeでこのリポジトリを開く
2. 当日の日付ファイルを作成する
3. テンプレートに沿って3〜5分で記入する
4. 保存する
5. GitHub DesktopでCommitする
6. GitHubへPushする

## 今後追加したい機能

- 今日の日付ファイルを自動生成する機能
- テンプレートを自動挿入する機能
- コミットメッセージを自動生成する機能
- GitHubへのPushを半自動化する機能
- ChatGPTや音声入力と連携してTILを自動作成する機能
